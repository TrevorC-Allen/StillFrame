# StillFrame Design System

## Product Feel

StillFrame is a quiet desktop tool. The UI favors scanning, repeated use, and predictable controls over a marketing-style layout.

## Layout

- Fixed sidebar for navigation and sources.
- Main workspace for browsing and list views.
- Right player panel for transport, seek, speed, audio, and subtitle controls.
- Repeated rows use compact 8px-radius surfaces.

## Color

- Base: near-black and neutral graphite.
- Accent: warm amber for progress and favorites.
- Secondary accent: muted teal for playback and file media icons.
- Error: restrained coral.

## Identity

- The StillFrame mark is a vector symbol built from a film frame and a play aperture.
- Do not use letter placeholders such as `SF`, `F`, or `P` as product or file icons.
- Use `/static/stillframe-mark.svg` for the Web MVP and `/stillframe-mark.svg` inside the Electron app.

## Controls

- Icon buttons for actions such as add, back, play, pause, stop, and favorite.
- Select menus for audio, subtitle, and speed choices.
- Range input for seek.
- Compact status chips for runtime availability.
